Methodology

Benchmark setup, environment, and measurement approach.

Compiler & Build

  • Crystal release-built compiler (LLVM backend)
  • --no-codegen flag (type-check + dependency only)
  • Incremental branch: incremental-compilation
  • 3 runs per scenario â€” min/median/max reported

Scenarios Defined

  • Stock: vanilla crystal build, no cache
  • One-shot: incremental build + cache write in one pass
  • Cold: cache written, memory cleared, re-read from disk
  • Warm: cache already in OS page cache â€” typical dev loop
  • Watch: touch src/main.cr then incremental rebuild

Platform

  • macOS (Apple Silicon â€” ARM64)
  • LLVM 21.1.8 at /opt/homebrew/Cellar/llvm/21.1.8_1/
  • All processes measured with time wall clock
  • Cache cleared between stock/cold runs via sync && purge
  • Frameworks installed via shards install
